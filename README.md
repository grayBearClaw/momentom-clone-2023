주요 기능

background, quotes: 저장된 이미지(5개)와 텍스트(10개)를 랜덤으로 1개씩 로딩

clock: 시간 표현

login: localstorage에 유저 정보 저장, 저장 유무에 따라 입력폼 또는 greeting 메세지 출력

todo: 유저가 입력한 값을 저장 및 삭제 가능

weather: openweathermap의 API를 활용하여 실시간 위치, 날씨, 기온 정보를 표현
